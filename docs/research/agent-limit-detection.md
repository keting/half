# Agent 限制与重置时间检测研究

状态: v0.3 纯研究。

## 目的

HALF 当前支持人工维护的 agent 可用性和重置时间元数据。此研究跟踪调查是否可以通过合规、可维护的方法获取任何 agent 限制或重置时间数据。

## 范围

研究可覆盖：

- Claude Code
- Codex
- Copilot
- GLM
- Kimi

对每个 agent 或平台，将可能方法分类为：

- 官方支持 API
- 用户可见人工输入
- 供用户检查的本机日志或 CLI 输出
- 不支持/私有接口
- UI 自动化或逆向工程

## v0.3 边界

v0.3 不得将自动限制或重置时间检测集成到产品中。输出为研究笔记和推荐。

HALF 不得依赖私有 API、UI 自动化、逆向工程或凭证共享来自动化订阅制 coding agent。

## 预期产出

最终研究笔记应包括：

- 按 agent/工具的可行性矩阵
- 合规风险
- 运营可靠性风险
- 推荐产品方向
- 后续版本的开放问题
