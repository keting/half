# ADR 0004: Agent 可见性模型

状态: proposed

## 背景

HALF 当前将 agent 视为所有者范围的资源。v0.3 计划在保持普通用户私有 agent 私有的同时，支持窄范围的管理员管理公共 agent 共享目录。

## 决策

v0.3 预期方向：

- 管理员管理的公共 agent 对所有用户可见
- 普通用户可以在允许的地方使用公共 agent
- 普通用户不能编辑管理员管理的公共 agent
- 普通用户的私有 agent 保持私有
- 此功能不得在 v0.3 扩展为通用资源共享

实现前应定稿确切的数据库字段和访问控制语义。

## 开放问题

- 公共可见性应显式表示在 agent 上还是从创建者角色派生？
- 角色变更应如何影响现有 agent？
- 哪些查询应使用可见 agent 范围 vs 可编辑 agent 范围？
- 公共 agent 的删除或禁用应如何处理跨用户项目引用？

## 备选方案

- 在 v0.3 保持所有 agent 仅所有者可用。
- 引入跨用户的通用资源共享。
- 允许普通用户发布公共 agent。

## 后果

此更改触及访问控制语义，应在实现前由核心维护者审查。
